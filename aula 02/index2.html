HTML Semântico

Semântica é dar significado à estrutura do documento. Isso melhora acessibilidade, SEO, manutenção e segue padrões do W3C.

Tags principais

  • <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>
  • <figure> + <figcaption>, <time>, <mark>

Boas práticas

  1. Use <main> uma vez por página.
  2. Mantenha alt descritivo em imagens.
  3. Relacione <label> com for.
  4. Use <header>/<footer> também dentro de seções/artigos quando fizer sentido.
